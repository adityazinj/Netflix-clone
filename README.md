## Installation / How to Run Locally
1. Clone the repo:
   ```bash
   git clone https://github.com/your-name/Netflix-clone.git

2. Navigate to the project folder:
    cd netflix-clone

3. Install dependencies:
    npm install

4. Start the development server:
    npm run dev

5. Open your browser and go to:
    http://localhost:5173
